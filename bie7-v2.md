## 一、项目概述

憋7（Sevens / Fan Tan）是一款4人纸牌游戏，使用一副52张标准扑克（去掉大小王）。玩家轮流将手牌放到桌面，最先出完者为胜，最终根据扣牌分数排名。

当前版本为**单机完整版**（1人类+3AI），支持**多轮累计积分制**和**换三张模式**。联机版云函数已预留。

## 二、已完成功能

### 2.1 核心玩法
- 标准憋7规则：洗牌、发牌、出牌（相邻接龙）、扣牌（无牌可出时）、结算
- Fisher-Yates 洗牌算法，52张牌随机分发4人各13张
- 持有7♥的玩家先手，桌面为空时只能出7
- 出牌规则：必须与桌面同花色相邻（点数差1），填补缺口同样有效
- 扣牌规则：无牌可出时必须扣牌，有牌可出时不可主动扣牌
- 结算：扣分 = 扣牌点数之和（A=1, J=11, Q=12, K=13），扣分最少者获胜

### 2.2 多轮累计制
- 每轮结束后显示本轮排名和扣分
- 总分数跨轮累积显示
- 对手头像旁显示累计分（红色标注）
- 结算弹窗可点"下一轮"继续或"结束游戏"退出

### 2.3 换三张模式
- 每轮开始前，人类玩家选择3张手牌
- 选中牌的点数和必须 ≤ 21
- 确认后与对家（电脑B）随机交换等量牌
- 预览换来的3张牌后进入出牌阶段

### 2.4 UI 设计
- 微信小程序简约风格，淡色背景
- 横屏布局，适配手机屏幕
- 牌面左上角显示点数和花色（真实扑克风格）
- 桌面两列布局：左列 ♠♥ / 右列 ♣♦
- 同花色牌堆叠展示（露出左侧点数花色），52张全出也能放下
- 手牌横向滚动，扣牌右侧并排
- 双击出牌快捷操作
- 选中牌金色高亮上浮
- 7号牌橙色边框锚点标记
- 新出牌弹出动画

### 2.5 AI 策略
- 优先出可出的最小牌（手牌已按花色→点数排序）
- 无牌可出时扣点数最大的牌
- 思考延迟 800~1400ms 模拟真人

## 四、关键数据流

```
game.js (页面状态)
  │
  ├─→ utils/game-engine.js  ← 纯函数，操作玩家对象和桌面Set
  │     ├─ hand.splice()     ← 修改手牌
  │     ├─ withheld.push()   ← 修改扣牌
  │     └─ table[Suit].add() ← 修改桌面
  │
  ├─→ syncState()            ← clonePlayers + setData
  │     │
  │     └─→ 触发组件 observer 和 WXML 重渲染
  │           ├─ player-area: 显示对手头像/名称/扣牌数/总分
  │           ├─ table-board: 两列牌阵渲染
  │           └─ card-cmp: 卡牌渲染
  │
  └─→ 用户交互/AI决策
        └─→ 再次调用 engine → syncState → 循环
```
